Set of tools to troubleshoot and debug j2ee applications

  • System, application, session and request parameters
  • Test http communication (firewalls issues)
  • Display log4j files
  • Reconfigure Log4j at runtime
  • Display MDSP Config
  • JDBC test
  • DataSource test (lookup a jndi data source and invoke it)
  • DataSource testCase : forget to close connections to see if the dataSource limits the number of opened connections)
  • JNDI lookup test
  • Generate a Java Core of an Ibm Jvm
  • Generate a Heap Dump of an Ibm Jvm
  • Generate a System Dump of an Ibm Jvm
  • Introspect class
  • Event broadcaster test
  • Jamon Api results page
  • JMX dump Platform MBeanServer
